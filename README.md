# ptcg-live-zh-mod

本项目皆在为 Pokémon TCG Live 桌面版 (Windows & macOS) 带来最完善的中文化体验。

## 参与翻译

如果你想参与翻译，可以前往 <https://paratranz.cn/projects/9617> 申请加入翻译项目并翻译文本。
